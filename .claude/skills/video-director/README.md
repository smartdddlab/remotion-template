# Video Director - 专业视频制作框架

---

## 简介

Video Director 是一个**技术无关**的专业视频制作框架，专注于将视频脚本转化为结构化的生产计划。它采用**纯设计语言**，让导演专注于**创意愿景（WHAT）**而非技术实现（HOW）。

### 核心理念

- **导演角色**：统筹创意愿景、情感节奏、视觉风格
- **技术无关**：不绑定任何特定技术栈（Remotion/After Effects/其他均可）
- **通用动画词汇**：使用标准化的设计语言描述动画和视觉
- **智能风格系统**：基于内容的自动风格识别与映射

### 适用场景

✅ **适合**：
- 有具体视频脚本或详细内容大纲
- 需要结构化生产计划（场景、音频、视觉设计）
- 需要跨部门协调（脚本、音频、视觉、后期）
- 需要技术无关的创意指导

❌ **不适合**：
- 仅有主题无具体内容（如"做个AI视频"）
- 需要实体拍摄团队
- 简单的社交媒体短片

## 核心特性

### 1. 结构化生产框架

基于导演视角的完整生产流程：

```
输入（脚本/内容）
    ↓
分析（需求、受众、风格）
    ↓
规划（创意元素、视觉风格、音频）
    ↓
协调（各部门一致性）
    ↓
输出（完整生产计划 + 质量检查点）
```

### 2. 智能风格系统（v5.2+）

四维度风格定义，自动映射到视觉设计：

| 维度 | 说明 | 示例 |
|------|------|------|
| **文化** | 地域文化传统 | 中国风、和风、欧美风 |
| **时代** | 历史时期审美 | 古典、现代、未来、复古 |
| **流派** | 艺术风格流派 | 极简、赛博朋克、蒸汽波 |
| **技术** | 视觉技术风格 | 电影感、平面设计、插画 |

### 3. 置信度驱动的风格处理

- **高置信度 (>80%)**：自动应用风格
- **中置信度 (50-80%)**：提供选项供选择
- **低置信度 (<50%)**：要求明确指定
- **冲突检测**：自动识别不合理组合并解决

### 4. 质量检查点系统

每个阶段设置审查关口：
- 脚本审批 → 音频制作前
- 音频审批 → 视觉制作前
- 视觉审批 → 后期制作前
- 最终审批 → 分发前

## 视频主题风格系统

### 文化风格维度

基于地域文化传统的视觉语言：

| 风格 | 核心元素 | 典型配色 | 适用场景 |
|------|---------|---------|---------|
| **中国风** | 水墨、祥云、龙凤、留白 | 黑白金、青绿、中国红 | 传统文化、文化科普、历史 |
| **和风** | 樱花、波浪纹、禅意几何 | 樱花粉、蓝绿、大地色 | 治愈、冥想、放松内容 |
| **欧美风** | 摄影质感、几何构图 | 高饱和、互补色 | 商业、现代感、时尚 |
| **韩风** | 温柔典雅、现代融合 | 粉彩、白金、大地色 | 美妆、生活方式、温柔主题 |
| **阿拉伯风** | 几何纹样、金箔装饰 | 蓝金、绿银、红铜 | 异域文化、宗教主题 |
| **非洲风** | 部落纹样、大地色彩 | 赭石、橙黄、深绿 | 自然、原始、部落文化 |
| **拉丁美洲风** | 鲜艳色彩、民间艺术 | 鲜艳红黄、蓝绿 | 热情、节日、民俗 |

### 时代风格维度

基于历史时期的视觉审美：

| 风格 | 核心特征 | 典型元素 | 适用场景 |
|------|---------|---------|---------|
| **古典** | 对称、雕刻、油画质感 | 深红、金、棕、古典建筑 | 历史、文化、庄重主题 |
| **现代** | 几何、简洁、无衬线 | 白灰黑+单色强调 | 商业、科技、专业内容 |
| **未来** | 霓虹、全息、粒子效果 | 霓虹粉、赛博青、深黑 | 科技、AI、科幻主题 |
| **复古** | 胶片颗粒、老物件 | 复古棕、暖黄、褪色感 | 怀旧、回忆、历史重现 |
| **蒸汽朋克** | 齿轮、蒸汽、黄铜机械 | 棕黄、铜色、灰 | 科幻、异世界、机械美学 |

### 流派风格维度

基于艺术流派的设计风格：

| 风格 | 核心特征 | 视觉元素 | 适用场景 |
|------|---------|---------|---------|
| **极简** | 大量留白、几何形状 | 单色+1-2强调色 | 教育、商业、治愈、高端 |
| **赛博朋克** | 霓虹灯、全息、故障艺术 | 霓虹粉、赛博青、深黑 | 科技、AI、未来、叛逆 |
| **蒸汽波** | 故障艺术、90年代元素 | 粉紫、蓝绿、橙 | 怀旧、艺术、亚文化 |
| **波普** | 波点、重复图案、高对比 | 红蓝黄等高饱和对比 | 流行文化、艺术、年轻 |
| **抽象** | 非具象形状、色彩碰撞 | 大胆色彩组合 | 艺术表达、概念传达 |
| **超写实** | 照片级真实感 | 真实光影、细节丰富 | 产品展示、高端质感 |
| **卡通** | 夸张表现、粗线条 | 高饱和度、明亮 | 儿童、趣味、轻松内容 |

### 技术风格维度

基于视觉技术实现的风格：

| 风格 | 特征描述 | 技术要点 |
|------|---------|---------|
| **电影感 (Cinematic)** | 电影级画面、浅景深、调色 | 镜头语言、光影、色彩分级 |
| **平面设计 (Graphic)** | 扁平化、排版驱动、色块 | 网格系统、字体、对比 |
| **插画 (Illustrative)** | 手绘感、艺术化、叙事性 | 笔触、纹理、故事感 |
| **极简 (Minimalist)** | 少即是多、留白、克制 | 负空间、比例、呼吸感 |
| **科技 (Tech)** | 数据可视化、HUD、线条 | 几何、数字感、冷色调 |
| **超写实 (Hyper-realistic)** | 超越真实的细节和质感 | 微观细节、完美光影 |

### 风格组合策略

#### 推荐组合

**现代中国风**（文化+时代+流派）
```yaml
cultural_style: "chinese"
era_style: "modern"
genre_style: "minimalist"
```
- **视觉特征**：水墨留白 + 现代几何 + 传统纹样
- **配色**：黑白金为主，青绿点缀
- **适用**：文化科普、传统现代融合、高端文化

**赛博朋克未来**（时代+流派）
```yaml
era_style: "future"
genre_style: "cyberpunk"
```
- **视觉特征**：霓虹灯、全息投影、故障艺术
- **配色**：霓虹粉、赛博青、深黑背景
- **适用**：AI科普、科技产品、未来主题

**和风治愈**（文化+流派）
```yaml
cultural_style: "japanese"
genre_style: "minimalist"
```
- **视觉特征**：樱花元素、禅意留白、自然质感
- **配色**：樱花粉、蓝绿、大地色
- **适用**：冥想、放松、治愈内容

#### 冲突避免

以下组合会产生冲突，系统自动警告并调整：

- ❌ **中国风 + 赛博朋克**：传统 vs 未来（冲突）
- ❌ **古典 + 蒸汽波**：古典 vs 波普（冲突）
- ❌ **极简 + 蒸汽朋克**：极简 vs 蒸汽（冲突）

**冲突解决优先级**：
```
文化风格 > 时代风格 > 流派风格 > 技术风格
```

必须有具体内容，拒绝仅主题/标题的请求：
- ✅ 完整脚本/逐字稿
- ✅ 详细场景大纲
- ✅ 核心内容要点

### 使用流程

1. **查看快速开始指南**：`docs/quick-start-guide.md`
2. **了解导演决策框架**：`docs/director-decision-framework.md`
3. **参考完整模板**：`docs/complete-template-reference.md`
